Below you will find all the parts necessary. Rough cost:

  • Raspberry Pi: $50
  • SD Card: $10
  • Monitor: $250
  • HDMI cable: $10
  • Raspberry Pi Case: $15
  • Total: $335

Bill of Materials

Required

Raspberry Pi

Amazon

Can be found for cheaper elsewhere, but not always in stock

SD Card

Amazon

Any 8GB or bigger SD card should work. Highly preferable to use a class 10 card.

HDMI Cable

Amazon

This is a cheap and effective version. Could use something quite a bit shorter. Also worth checking out monoprice.com

Monitor

Amazon

Any monitor with HDMI should work. We have verified the above one works well and is fairly bright. If using above, disable sleep.

Raspberry Pi Case

Amazon

Optional

Wireless Dongle

Amazon

Fully compatible out of the box with the Raspberry Pi. Will require configuration.
